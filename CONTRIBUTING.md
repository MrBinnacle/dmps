# Contributing to DMPS

## Development Setup
```bash
git clone https://github.com/MrBinnacle/dmps.git
cd dmps
pip install -e .
pip install pytest
```

## Running Tests
```bash
python -m pytest tests/ -v
```

## Code Style
- Follow PEP 8
- Use type hints
- Add docstrings for public functions

## Submitting Changes
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Submit a pull request
